<h1>Installing</h1>

**Clone Repository**
  1. git clone https://github.com/callmeSlikker/Serial-Test.git
  2. cd Serial-Test

**Install Backend**
  1. cd backend
  2. npm install

**Install Frontend**
  1. cd frontend
  2. npm install

<h1>Running</h1>

**Stay in folder Serial-Test Ex: PS C:\Users\Intern\Serial-Test>**
  1. npm run start
